Product Updates

  • Bandwith Pricing calculator: our new tool that will help you estimate overage charges due to outbound transfer from your DigitalOcean resources.

  • New 1-Click Apps in DigitalOcean Marketplace: our new tool that will help you estimate overage charges due to outbound transfer from your DigitalOcean resources.
    • A curated collection of 1-Click Apps for working from home or remotely
    • Strapi a leading open source Node.js Headless CMS
    • Kepler a powerful drag & drop Wordpress website builder
    • Dokos a full-featured open source ERP
    • Jelastic PaaS automates creation, scaling, clustering, and security updates of microservices and monolithic apps
    • Titra an easy and modern way to track your time spent on projects

  • Billing history endpoint now available in DigitalOcean API: Along with invoices and balance endpoints released earlier this year, our third wave of billing API improvements now adds the billing_history endpoint that allows you to access past invoiced amounts, payments made, credits granted, etc. Check out the docs for the billing history endpoint and let us know what you think.

  • Deletion of associated resources: When you destroy a Droplet, you can now easily delete associated resources like snapshots and volumes.

  • Managed Databases for MySQL: As of MySQL 8.0, the default authentication plugin was changed to caching_sha2_password. MySQL 5.7 used mysql_native_password as the default. If you’re running into challenges with authenticating existing clients and applications, please update your user authentication method.

See more minor updates in the DigitalOcean release notes.

New Tutorials

  • If you love developing locally with VS Code but want to run program execution or unit tests on a remote server, follow Mason Egger’s guide, How to use Visual Studio Code for Remote Development via the Remote-SSH Plugin.

  • Learn the difference between yield and return inside a generator and more about these JavaScript functions in Tania Rascia’s tutorial, Understanding Generators in JavaScript.

  • As you scale your apps with DigitalOcean Kubernetes, it’s important to consider security. Recommended Steps to Secure a DigitalOcean Kubernetes Cluster by Damaso Sanoja will walk you through the fundamental steps you should take to secure your cluster.

Have you considered writing a tutorial that will help others in the developer community? Check out our Suggested Topics for Tutorials to see what our Editorial team is looking for!
